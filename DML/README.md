<b>DML – Data Manipulation Language (Linguagem de Manipulação de Dados)</b>
<p></p>  
A DML é utilizada para manipular os dados armazenados no banco. Ela permite inserir, atualizar, excluir e consultar registros em tabelas.
<p></p>  

Exemplos de operações:
<ul>
<li>Inserção de registros</li>
<li>Atualização de dados existentes</li>
<li>Remoção de registros</li>
<li>Consulta de dados</li>
</ul>
